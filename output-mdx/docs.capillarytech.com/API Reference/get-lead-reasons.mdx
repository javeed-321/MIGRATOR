## Get Lead Reasons

Retrieves the list of reasons added for the org with reason ids. These reason ids are required to create or update leads.

## Path parameter

| Parameter | Description            |
| :-------- | :--------------------- |
| leadId    | Unique ID of the lead. |

## Response parameter

| Parameter | Description                                    |
| --------- | ---------------------------------------------- |
| id        | Unique identifier for the reason.              |
| reason    | Description or details of the specific reason. |
