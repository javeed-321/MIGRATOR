## Edit or Disable a Cart Promotion

You can manage the lifecycle of your promotions through the centralized list view. This allows you to update campaign durations, adjust creative messaging, remove underperforming offers, or reactivate a previously deactivated promotion.

## 1. Navigation

1. Go to **Engage+ > Campaigns**.

<br />

<Image src="https://files.readme.io/e9f6e051ff9ceaed42c4d0d6c8d78d9c9df6d2a82bf18ef67eb03bdd8e3b54cf-Screenshot_2026-03-31_at_5.35.35_PM.png" width="1920" height="1080" alt="" />

2. Select the specific **Campaign** containing your promotion.

<br />

<Image src="https://files.readme.io/247a1aad4983937d4f8da8a76ae4ae2ff2cdbdfe37a3fbe264ca883de9e520ef-Screenshot_2026-03-31_at_5.38.33_PM.png" width="100%" height="1080" alt="" />

3. Navigate to the **Promotions** tab to view the list of all created cart promotions.

<Image src="https://files.readme.io/f98846b73310c18296349f7c1158bf30685c5b38cd7574d94de2660e8e65d35b-Screenshot_2026-03-31_at_5.42.45_PM.png" width="1920" height="1080" alt="" />

## 2. How to edit a promotion

If you need to update the details of an existing offer:

1. Locate the promotion in the list.
2. Click the **Three-Dot Menu (⋮)** at the end of the promotion row.
3. Select **Edit**.

<Image src="https://files.readme.io/7ad711ad233fd8fb8290a2f703a5d224c43638dacbf95e48242cd54279d6eb4c-Screenshot_2026-05-07_at_7.41.44_PM.png" width="1920" height="1080" alt="" />

5. Modify the necessary fields (for example, Description, End Date, or Communication Templates).
6. Click **Update** to apply the changes.

<Callout kind="alert">

  While most fields are editable, **Redemption Restrictions** (such as individual customer limits or global counts) **cannot be updated** once the promotion is live.

</Callout>

## 3. How to deactivate a cart promotion

If a promotion is no longer required or was created in error:

1. Locate the promotion in the list.
2. Click the **Three-Dot Menu (⋮)**.
3. Select **Deactivate**.

<Image src="https://files.readme.io/ad2b3c703fa4164e3d15ed295785610bc91ff841e35c45f1c452871125e38283-Screenshot_2026-05-07_at_7.42.20_PM.png" width="1920" height="1080" alt="" />

4. A confirmation prompt appears. Select **Yes** to disable the promotion.

## 4. How to activate a cart promotion

You can reactivate a deactivated promotion from the same promotions list. The **Activate** option appears in the three-dot menu only when a promotion is in an inactive state.

1. Locate the deactivated promotion in the list.
2. Click the **Three-Dot Menu (⋮)** at the end of the promotion row.
3. Select **Activate**.

<Image src="https://files.readme.io/placeholder-activate-cart-promotion.png" width="1920" height="1080" alt="" />

4. A confirmation prompt appears. Select **Yes** to activate the promotion.

A success notification confirms that the promotion has been activated. The promotion list refreshes automatically to reflect the updated status.

<Callout kind="alert">

  Promotions that have already ended (past their end date) cannot be reactivated. The **Activate** option will not appear for such promotions.

</Callout>
