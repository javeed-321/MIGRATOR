# Organization

At Capillary, organizations (also referred to as 'orgs' in this document) are managed in three ways, based on the brand structure.

<Image align="center" border={true} width="65% " src="https://files.readme.io/498097a4f9467c3f2844e368863b6d929702f8d24cfdcd7532cd18a0f69cd5d3-image.png" className="border" />

The table below lists the types of organizations, what it is, and the use cases that which they are suitable.

| Type of Org                                                                                                                                                                                                                 | What it is                                                                                                                                                          | Who is it suitable for?                                                                                                                                                             |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **<Anchor label="Independent / Standard Organization" target="_blank" href="https://docs.capillarytech.com/docs/organization-management#/">Independent / Standard Organization</Anchor>**                                   | A standalone organization that manages its own entities, customer data, loyalty programs, and engagement initiatives without any dependency on other organizations. | Businesses running independent loyalty programs where data sharing across entities is not required.                                                                                 |
| **<Anchor label="Organization Units (OUs) within a single organization" target="_blank" href="https://docs.capillarytech.com/docs/organisation-units-ou#/">Organization Units (OUs) within a single organization</Anchor>** | A single organization set up for a conglomerate, where each brand is mapped to a dedicated OU.                                                                      | For conglomerates that want to manage loyalty operations across all their brands centrally.                                                                                         |
| **<Anchor label="Connected Organizations" target="_blank" href="https://docs.capillarytech.com/docs/connected-organizations#/">Connected Organizations</Anchor>**                                                           | Each brand under the conglomerate is set up as an independent organization in the Capillary platform.                                                               | For conglomerates that want to manage loyalty operations across all their brands centrally, while also allowing each brand to have its own loyalty program and manage its own data. |
