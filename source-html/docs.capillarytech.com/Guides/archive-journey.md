# Archive journeys

Archive journeys to remove them from the default journey list. Archived journeys are hidden from the default view but remain accessible in reports, exports, and delivery logs. You can unarchive a journey at any time.

> **Note:** Journey archival is not enabled by default. Create a Jira ticket to the PST team to enable it for your organization.

## Archive a journey

Journeys must be in an eligible status before you can archive them. Active journeys and those in **Awaiting Approval** status can't be archived.

> **Note:** A journey's archival eligibility is based on its overall status, not whether individual versions have been stopped. A journey with a stopped version can still be Active if its configured end date hasn't passed.

To archive a journey, follow the given steps below:

1. Go to **Engage+** > **Journeys**.
2. Find the journey you want to archive.
3. Select the actions menu (⋮) for that journey.
4. Select **Archive**.
5. In the confirmation dialog, select **Archive** to confirm.

The journey is removed from the default journey list and moved to the archived view. A success notification appears: *Journey archived*.

## Bulk archive journeys

To archive bulk journey, follow the given steps below:

1. Navigate to **Engage+** > **Journeys**.
2. Select the checkboxes next to the journeys you want to archive. Select up to 10 journeys at a time. If your selection includes Active or Awaiting Approval journeys, the **Archive** button remains disabled. Deselect ineligible journeys before proceeding.
3. Select **Archive**.
4. In the confirmation dialog, review the count, then select **Archive** to confirm.

A success notification shows the number of journeys archived.

> **Note:** The bulk archive limit is 10 journeys per operation. To archive more, deselect your current selection and repeat the process.

## View archived journeys

Archived journeys don't appear in the default journey list. To view them:

1. Navigate to **Engage+** > **Journeys**.
2. In the status filter, select **Archived**.

You can search and filter archived journeys the same way you would active ones.

\[Screenshot placeholder: Journey listing page with the Archived filter applied]

## Unarchive a journey

To unarchive a journey, follow the given steps below:

1. Navigate to **Engage+** > **Journeys**.
2. In the status filter, select **Archived**.
3. Find the journey you want to unarchive.
4. Select the actions menu (⋮) for that journey.
5. Select **Unarchive**.
6. In the confirmation dialog, select **Unarchive** to confirm.

The journey returns to the active journey list. A success notification appears: *Journey unarchived*.

\[Screenshot placeholder: Archived journey list showing the Unarchive option in the row actions menu]

## Bulk unarchive journeys

To bulk unarchive a journey, follow the given steps below:

1. Navigate to **Engage+** > **Journeys**.
2. In the status filter, select **Archived**.
3. Select the checkboxes next to the journeys you want to unarchive.
4. Select **Unarchive**.
5. In the confirmation dialog, select **Unarchive** to confirm.

A success notification shows the number of journeys unarchived.
