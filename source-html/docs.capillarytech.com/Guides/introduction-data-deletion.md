# Introduction

The data deletion feature allows customers to delete their personal data from the customer data platform. This provides customers greater control over their privacy and helps to protect their data or unauthorised access to their information.

### **Typical examples**

* Name
* Mobile number
* Email address
* External customer ID
* Any identifier that uniquely links data to a person
