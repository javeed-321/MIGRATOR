# Rate limits

This page provides you information regarding the rate limit of the APIs.

The rate limit is the number of API calls that can be made in a minute, which is configured in the back end. This reduces the load on servers and stops certain fraud activities.

The rate limits vary for each API and are dependent on the region.

* **Demo and Testing Clusters:** 100 requests per minute per API key

* **Other Organizations:** The rate limit is specific to each brand.

To modify the limit, create a ticket with the Capillary Product support team.
