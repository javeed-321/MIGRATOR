# PII deletion

Personal Identifiable Information is defined as any information that can be used to identify someone.

The Personally Identifiable Information (<Glossary>PII</Glossary>) data deletion feature allows customers to delete their personal data from the customer data platform. This gives customers greater control over their privacy and helps protect their data or unauthorised access to their information.

For complete information on the PII configuration process, refer to the documentation <Anchor label="here" target="_blank" href="https://docs.capillarytech.com/docs/pii-configuration#/">here</Anchor>.
